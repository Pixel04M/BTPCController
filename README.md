# BT PC Controller

> Turn your Android phone into a wireless Bluetooth keyboard — no cables, no dongles, no extra software.

Ever wanted to control your PC from across the room? Or prank a friend by typing on their computer without touching it? **BT PC Controller** connects your phone to any Bluetooth-enabled device and lets you send real keystrokes instantly.

---

## What it does

- Pairs with your device over **Bluetooth HID** (the same protocol real keyboards use)
- Sends **modifier keys** — Ctrl, Shift, Alt, Win/Cmd — just like a physical keyboard
- Supports up to **6 simultaneous key presses**
- Works with **no setup required** on the host — it shows up as a standard keyboard

## Works with

| Platform | Supported |
|----------|-----------|
| Windows  | Yes |
| Linux    | Yes |
| macOS    | Yes |
| Android  | Yes |


## Requirements

- Android phone with Bluetooth
- A target device with Bluetooth enabled and discoverable

---

Built with Kotlin using the Android Bluetooth HID profile.
